Este repositorio implementa un módulo web representativo para un sistema universitario de gestión de tareas. Incluye un backend en Flask + SQLite y un frontend en React, donde un catedrático puede crear y publicar tareas y un estudiante puede visualizar únicamente las tareas publicadas. El proyecto incorpora un pipeline de calidad que ejecuta pruebas automáticas, genera cobertura, realiza análisis estático y envía el código a SonarCloud para evaluar mantenibilidad, confiabilidad y seguridad antes de integrar cambios.
